# Pendahuluan
 
analista101 adalah sebuah platform yang menyediakan tutorial, tips, dan trik seputar analisis data. Platform ini dibuat dengan tujuan untuk membantu teman-teman analista apabila sedang mengalami kesulitan dalam menganalisis data.

## Cara Berkontribusi

1. Fork repository ini
2. Clone repository yang sudah di-fork
3. Buat branch baru dengan nama yang deskriptif
4. Lakukan perubahan
5. Commit perubahan yang sudah dilakukan
6. Push branch ke repository yang sudah di-fork
7. Buat pull request ke repository ini
8. Tunggu pull request di-review oleh owner repository

## Cara Menjalankan Project

```bash
# install dependencies
$ npm install # or pnpm install or bun install
```
    
```bash
# serve with hot reload at localhost:3000
$ npm run dev # or pnpm run dev or bun dev
```
